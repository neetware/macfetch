## Requirements:

- [Swift 5.0](https://swift.org/download/)

## Installation:

### Build and add it to your $PATH 

```bash
swiftc macfetch.swift -o macfetch
sudo mv macfetch /usr/local/bin
```
## Usage:
You can now run it with

```bash
macfetch
```
